# Linux

Как перезагрузить сервер:

```shel
sudo reboot
```

или

```shell
sudo shutdown -r now
```

Последний вариант предпочтительнее, так как позволяет указать причину перезагрузки

```shell
sudo shutdown -r now 'System maintenance'
```
