title: Вызов функции по ее строковому имени в python

```python
import foo
method_to_call = getattr(foo, 'bar')
result = method_to_call()
```

или для класса

```python
class C:
    def m(self):
        return "result"

an_object = C()

class_method = getattr(C, "m")
result = class_method(an_object)
```
