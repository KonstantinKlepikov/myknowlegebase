## Python3

Фолдер, в котором запущен скрипт:

```python
import pathlib
pathlib.Path(__file__).parent.resolve()
```

Текуший рабочий фолдер:

```python
import pathlib
pathlib.Path().resolve()
```

## Python3 или python2

Фолдер, в котором запущен скрипт:

```python
import os
os.path.dirname(os.path.abspath(__file__))
```

Текуший рабочий фолдер:

```python
import os
os.path.abspath(os.getcwd())
```
